<task name="Add server endpoints for audio control">
<read_first>
  src/server/routes/ptz.ts
</read_first>
<action>

1. Add to `src/server/routes/ptz.ts` (or create `src/server/routes/audio.ts`):

   ```typescript
   // GET /api/audio/capability
   router.get('/capability', async (ctx) => {
     const hasAudio = await camera.getAudioCapability();
     ctx.body = { hasAudio };
   });

   // POST /api/audio/volume
   router.post('/volume', async (ctx) => {
     const { volume } = ctx.request.body;
     await camera.setVolume(volume);
     ctx.body = { success: true };
   });

   // POST /api/audio/mute
   router.post('/mute', async (ctx) => {
     const { muted } = ctx.request.body;
     await camera.setMute(muted);
     ctx.body = { success: true };
   });
   ```

2. If camera doesn't support audio API: return `{ hasAudio: false }` gracefully

</action>

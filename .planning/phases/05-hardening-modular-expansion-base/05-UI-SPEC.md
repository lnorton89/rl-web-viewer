| Property | Value |

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Inline label offsets, compact micro-spacing |
| sm | 8px | Button groups, badge rows, compact stacks |
| md | 16px | Default control padding and card internals |
| lg | 24px | Section padding, card gaps, desktop layout rhythm |
| xl | 32px | Viewer overlay padding and major in-card emphasis |
| 2xl | 48px | Mobile shell padding and section separation |
| 3xl | 64px | Desktop shell top padding and page-level breathing room |

Exceptions: interactive controls must keep a minimum 44px height; PTZ direction and preset buttons stay 56px for hold/release accuracy; viewer frame remains content-led at 48vh-70vh instead of tokenized vertical spacing.

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #12161b | App background, viewer shell, overlay base surfaces |
| Secondary (30%) | #1d242c | Cards, support panels, diagnostics, PTZ, settings sections |
| Accent (10%) | #63c7b2 | Live-state indicators, verified state, focus ring, retry CTA, active mode/preset states |
| Destructive | #c65a5a | Stop camera, failed states, field errors, destructive emphasis only |

Accent reserved for: live status dots, `Retry Live View`, active mode chips, verified settings badges/callouts, focus-visible outlines, and active PTZ/preset states. Do not use accent for all buttons or neutral chrome.

---

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Retry Live View |
| Empty state heading | No settings available |
| Empty state body | This camera profile did not expose a safe settings section here. Refresh the dashboard after capability detection completes. |
| Error state | Live view failed. Show the short cause if known, then direct the user to retry live view or refresh the dashboard after capability detection completes. |
| Destructive confirmation | Stop Camera: no modal. Use an immediate action button, disable it while stopping, and announce the resulting status in the PTZ live region. |
